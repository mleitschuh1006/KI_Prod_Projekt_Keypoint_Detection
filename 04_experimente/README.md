# 04_experimente

Dieser Ordner macht die Experimente nachvollziehbar.

## Struktur

```text
04_experimente/
├── configs/   # YAML-Dateien mit Datensatz, Modell, Hyperparametern und Pfaden
└── runs/      # Protokolle einzelner Läufe, z. B. Logs oder kurze Markdown-Berichte
```

Jedes Experiment sollte eine eigene Config-Datei und einen kurzen Ergebnisbericht erhalten.
