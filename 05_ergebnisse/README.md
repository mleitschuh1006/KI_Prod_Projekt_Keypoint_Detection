# 05_ergebnisse

Dieser Ordner enthält alle Ergebnisse, die später in Bericht oder Präsentation übernommen werden.

## Typische Inhalte

- Tabellen mit Metriken
- Plots
- Beispielvisualisierungen erkannter Keypoints
- qualitative Vergleiche
- finale Bewertung der ausgewählten Methoden

Große Modellgewichte sollten nicht direkt ins Repository geladen werden. Stattdessen sollten Pfade oder externe Speicherorte dokumentiert werden.
